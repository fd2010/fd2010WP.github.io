    JQuery

    • Query is a fast, small, and feature-rich JavaScript library designed to simplify client-side scripting and make it easier to work with HTML documents, handle events, create animations, and perform various other tasks on web pages.

    Some key features and functionalities of jQuery include:

    • Javascript simplification
    • DOM manipulation
    • Event handling
    • AJAX support
    • Animations
    • Cross-browser compatibility
    • Plugin system
    • Chaining

    Checking that the doccument is ready

    before running JQuery you usually want to make sure the doccument has ran everything else first and is ready. You can do this by doing:

    • $(document).ready(...)

    Above is a jQuery function that waits for the HTML document to be fully loaded before executing the enclosed code.

    WHAT HAPPENS IF WE DON'T CHECK IF THINGS ARE READY?

    1. Code doesn't work (90% of time might throw an error)
    2. Code works .. the page loads, next page loads, then fails (**unreliable**)
